# Calculator CLI v2.1.x — Solver

**Theme:** Ask a question. Get an answer.

## Overview

Adds equation solving. `calc solve 'x^2 - 4 = 0' x` returns `x = 2, x = -2`;
`calc solve 'sin(x) = 0.5' x` returns a list of solutions in a given domain.
Exact solutions are returned when the form is detectable (linear, quadratic,
cubic); numerical solutions via Newton's method for general expressions.
With complex numbers from v1.3.x, `calc solve 'x^2 + 1 = 0' x` returns `x = i,
x = -i`. This is the version where the calculator stops being a tool for
evaluating known expressions and starts being a tool for finding unknowns.

---

## Success Criteria

- [ ] `calc solve 'x^2 - 4 = 0' x` prints `x = 2\nx = -2` to stdout and exits 0
- [ ] `calc solve '2*x + 3 = 7' x` prints `x = 2` and exits 0
- [ ] `calc solve 'x^2 + 1 = 0' x` prints `x = 0+1i\nx = 0-1i` and exits 0
- [ ] `calc solve 'sin(x) = 0.5' x --domain '0,2*pi'` prints `x = 0.5235987755982988\nx = 2.617993877991494` and exits 0
- [ ] `calc solve 'x^2 - 4 = 0' x --latex` prints LaTeX-formatted solutions and exits 0
- [ ] `calc solve 'x + y = 5' x` treats `y` as a parameter and prints `x = 5 - y` (symbolic, linear case) and exits 0
- [ ] `calc solve 'sin(x) = 2' x` prints `error: no real solutions` to stderr and exits 1
- [ ] `calc solve 'x^2 = x^2' x` prints `error: identity (all values satisfy the equation)` to stderr and exits 1
- [ ] `make test` passes clean

## Scope

**Included:**

- `calc solve '<lhs> = <rhs>' <variable>` subcommand
- Exact solving: linear (`ax + b = c`) and quadratic (`ax^2 + bx + c = 0`) forms
- Numerical solving: Newton's method for general expressions; configurable domain
  via `--domain 'min,max'`
- Complex solutions: when v1.3.x complex numbers are available, complex roots
  of quadratics returned automatically
- Symbolic linear solve: `x + y = 5` with `y` as free parameter returns `x = 5 - y`
- `--latex` output: solutions formatted as LaTeX
- Multiple solutions: all solutions found in the domain are returned, one per line
- `--n` flag: max number of numerical solutions to return (default: all found)

---

## Failure Modes

**No solution:**
- No real solutions exist → `error: no real solutions` to stderr, exit 1
- No solutions in specified domain → `error: no solutions in domain [min, max]` to stderr, exit 1

**Identity:**
- Equation is an identity (all values satisfy it) → `error: identity (all values satisfy the equation)` to stderr, exit 1

**Numerical:**
- Numerical solver fails to converge → `error: solver did not converge` to stderr, exit 1

---

## Interface Contracts

**CLI:**

```
calc solve '<lhs> = <rhs>' <variable> [--domain '<min>,<max>'] [--n <max-solutions>] [--latex]
```

| Flag | Default | Purpose |
|------|---------|---------|
| `--domain` | `[-1000, 1000]` | Search domain for numerical solutions |
| `--n` | unlimited | Max solutions to return |
| `--latex` | off | Format solutions as LaTeX |

**Output:** One solution per line in the form `<variable> = <value>`.

| Stream | Content |
|--------|---------|
| stdout | Solutions, one per line |
| stderr | `error: <description>` on failure |
| exit 0 | At least one solution found |
| exit 1 | No solution, identity, or solver failure |

---

## Constraints

- Depends on v2.0.x (Symbolic Differentiation) for the CAS representation
  that numerical solving uses for derivative computation (Newton's method requires `f'(x)`)
- Depends on v1.3.x (Complex Numbers) for complex root output
- Exact solving covers linear and quadratic forms only; cubic and higher require
  numerical methods
- The solver does not guarantee finding all numerical solutions; it finds
  solutions in the configured domain using a combination of bracketing and Newton's method

---

## Key Unknowns

1. **[P2]** Does Newton's method — which requires the derivative `f'(x)` at each
   step — compose cleanly with the symbolic differentiator from v2.0.x, or does
   numerical solving require a separate numerical differentiation path for cases
   where the symbolic derivative is too expensive or unavailable? The answer
   determines whether the solver depends on v2.0.x or can operate independently.

---

## Non-Goals

- Systems of equations (multiple equations, multiple unknowns)
- Inequality solving (`x^2 < 4`)
- Cubic and quartic exact solutions
- Differential equation solving
- Windows support
